Proyectos destacados

🧪 SimpleDEX

DEX funcional con lógica de liquidez y swap de tokens basada en contratos inteligentes.

Implementa el estándar ERC-20 para gestión de tokens fungibles.

El contrato está verificado en Sepolia y diseñado con arquitectura modular para facilitar la escalabilidad y auditoría.

Visualización desarrollada en React para interacción directa con el contrato.

🧪 Swap Detallado

Calcula el ratio de intercambio con la fórmula x · y = k.

Valida saldo, aprueba tokens y ejecuta el swap en Sepolia.

Arquitectura modular para facilitar auditoría y escalabilidad.
